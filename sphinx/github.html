GitHub とは¶

GitHub は Git を使用できるプロジェクトホスティングサービスです。 SourceForge や Google Code に似ています。 Ruby on Rails および Erlang で開発されていて、現在は GitHub 社によって保守されています。 最初 Chris Wanstrath, PJ Hyett, Tom Preston-Werner らによって開発されていました。 Ruby や Rails 関連を中心に FOSS のプロジエクトがたくさん登録されており、ワンクリックで簡単に fork することができます。 100MB までは無料で使用できます。
