更新日付:November 22, 2012

Sphinx とは¶

Sphinx は reStructuredText (reST) 形式のファイルを HTML ファイルに変換するドキュメント作成ツールです。 HTML の他に PDF, Epub, man, Tex など、たくさんのファイル形式に変換することができます。 最初は Python のドキュメントを作成するために作られたので、当然、Python で作成されています。 Georg Brandl によって開発され、BSDライセンスのもとで公開されています。
