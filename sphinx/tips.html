Create Link¶

Chromium の Extension です。 Firefox の addon Make Link の Chrome 版です。 WebサイトのタイトルとURLを色んな形式でコピーすることができます。 Sphinx や Redmine などリンクを独特な形式を登録しておくと便利です。

  • Create Link
Name Format Filter
Sphinx `%text% <%url%>`_  
Redmine “%text%”:%url%
