Amazon Web Services で Ubuntu Server を使用する¶

更新日付:November 21, 2012

はじめに¶

Ubuntu Server を早く簡単に使用してみたい場合、Amazon Web Services (AWS) の Amazon Elastic Compute Cloud(Amazon EC2) を使用するのが一番です。 Ubuntu 公式の Amazon Machine Image(AMI) が用意されておりポチッとするだけで始められます。 また、新規ユーザーには一年間の無料使用枠もあるのでコストを考えずに使用することができます。

  • クラウドサービス無料 利用枠のご案内 | アマゾン ウェブ サービス（AWS 日本語）
  • AWS 無料使用範囲とは - 入門ガイド

はじめ方¶

  • Sphinxをはじめよう :: Sphinx-users.jp

Ubuntu には sphinx のパッケージが用意されていて、簡単にインストールすることができます。

$ sudo apt-get install python-sphinx

Ubuntu 12.04 では sphinx 1.1.3 がインストールされます。

$ dpkg-query -l | grep python-sphinx
ii  python-sphinx  1.1.3+dfsg-2ubuntu2  documentation generator for Python projects (implemented in Python 2)

Sphinx では、以下のコマンドを実行し質問に答えることで、プロジェクトが作成されます。

$ sphinx-quickstart

いくつか質問されますが、必須で回答しなければならないのは、次の項目です。

  • プロジェクト名: n_nkmr’s wiki
  • バージョン番号: 0.0.1
  • 著者の名前: n_nkmr

後で、conf.py を編集することで変更が可能です。

  • sphinx-quickstartの詳細説明 :: ドキュメンテーションツール スフィンクス Sphinx-users.jp
