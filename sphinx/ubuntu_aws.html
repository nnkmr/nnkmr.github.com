Amazon Web Services で Ubuntu Server を使用する¶

更新日付:November 22, 2012

はじめに¶

Ubuntu Server を早く簡単に使用してみたい場合、Amazon Web Services (AWS) の Amazon Elastic Compute Cloud(Amazon EC2) を使用するのが一番です。 Ubuntu 公式の Amazon Machine Image(AMI) が用意されており、ポチッとするだけで始められます。 また、新規ユーザーには一年間の無料使用枠もあるのでコストを考えずに使用することができます。
