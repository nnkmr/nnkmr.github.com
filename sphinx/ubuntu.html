Ubuntu¶

更新日付:November 22, 2012

はじめに¶

Ubuntu を思えば長く使ってきました。 7.04 Feisty Fawn がリリースされる少し前に Vine Linux から乗り換えたので、5 年半ぐらい使っているようです。 Vine Linux から乗り換える前に仕事で Gentoo を使っていたのがいけなかったと思います。 当時は PC のマシンスペックも非力だったので、Gentoo はインストールだけでも何日もかかりました。 それでインストールが簡単な Ubuntu を使用するようになり、Vine Linux より最新のアプリケーションが使用できるので、常用環境も Ubuntu になりました。 Vine Linux にはあまり不満はなかったのですがね。
